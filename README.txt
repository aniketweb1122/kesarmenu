
# 🍽️ Kesar Telegram Order System (Deployable)

## Deploy on Render (Free)
1. Create a free account at https://render.com
2. Create new Web Service → Connect this repo/zip → Node.js
3. Build command: npm install
4. Start command: npm start

## Deploy on Heroku
1. Install Heroku CLI and login.
2. Run: heroku create kesar-telegram-order
3. Run: git push heroku main
4. Open your live site.

✅ When customer orders, details go directly to your Telegram bot!
